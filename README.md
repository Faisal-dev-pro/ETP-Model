# etp_model — Coupled Electro-Thermal-Pressure Model for 21700 Cell Thermal Runaway

MATLAB/Simulink implementation of a coupled electro-thermal-pressure (ETP)
model for prediction of internal gas pressure during thermal runaway in
21700 NMC lithium-ion cells, extending the Hatchard–Kim reduced-order
reaction kinetics framework.

Companion paper: *A Coupled Electro-Thermal-Pressure Model for Prediction
of Internal Gas Pressure Evolution During Thermal Runaway in 21700 NMC
Lithium-Ion Cells* (in preparation).

Validation dataset: Gulsoy et al. (2025), *Data in Brief*,
doi:10.1016/j.dib.2025.112190.

## Folder structure

```

             gulsoy_parsed.mat  (generated by convert_gulsoy_to_struct)

             etp_params_nmc.mat (generated by Phase 2 fit)

             etp_pure_matlab.m                  -- pure-MATLAB ode15s integrator
             run_phase1_sanity_check.m          -- Phase 1 sanity check (done)
             run_phase2_param_id.m              -- Phase 2 entry point
  results/   phase1_sanity.png, ...             -- generated figures & tables
  README.md
  LICENSE
```

## Quick start

```matlab
cd ~/MATLAB-Drive/etp_model
addpath('params','scripts','simulink');

% 1. Phase 1 baseline sanity check (no data needed)
run_phase1_sanity_check();
% -> results/phase1_sanity.png + console pass/fail report

% 2. Once you have data/TR_dataTable.mat in place:
convert_gulsoy_to_struct();          % parse + configure heaters + save
data = load('data/gulsoy_parsed.mat').data;

% Eyeball each test to confirm the parse is sensible
for i = 1:numel(data)
    plot_gulsoy_test(data(i), ...
        'savePath', sprintf('results/test%d_overview.png', i));
end

% 3. Phase 2 -- thermal-loss ID + cathode kinetics re-fit
results = run_phase2_param_id();
% -> params/etp_params_nmc.mat
```

## Source dataset structure (verified)

The Gulsoy 2025 `TR_dataTable.mat` exports as a 3-row wide table with
two sample rates per test:

| Field            | Source column   | Rate              |
|------------------|-----------------|-------------------|
| cell_id          | TestID          | 1 per test        |
| t_fast, P_internal, V_cell | ExpTime, IntPre, CellVoltage | ~10 kHz (~9.56M samples) |
| t, T_internal, T_surface_*, T_vent_* | ExpTimeTemp, MidIntTemp, ... | ~10 Hz (~9.6k samples) |

`heater_power` is **not** in the source data — the Methods section of
the paper specifies the heater protocol. `configure_gulsoy_heaters.m`
populates this field with a default 100 W constant profile, cut off
~10 s after runaway onset. Override per test if needed.

## Phase 1 verified output

A 30 W step input from 25 degC ambient produces (both MATLAB and Simulink
ode15s agreeing to four sig figs):

- Slow rise for ~9 min (Q_ext dominated)
- SEI → AnE → cathode → electrolyte cascade
- Peak 427.1 °C at t = 592 s, onset (dT/dt > 1 K/s) at 207.7 °C
- Cooldown completes inside the 30-min window

This is the canonical Hatchard-Kim LCO baseline. The Phase 2 NMC re-fit
will increase the peak toward the 600-800 °C range expected from NMC
chemistry.

## Modelling assumptions

- Single lumped thermal node (Bi < 0.3 for 21700)
- Four Hatchard–Kim reactions (SEI, AnE, cathode, electrolyte decomposition)
- Convective + radiative loss to constant ambient
- Pressure sub-model (Phase 3): single control volume V_int, ideal gas,
  gas production from electrolyte vapourisation + electrolyte decomposition
  + cathode O₂ release

## Reproducibility

- All parameters live in `params/etp_params.m`. Do not hard-code anywhere else.
- The Simulink model is built programmatically by `build_etp_model.m`,
  so the `.slx` is regenerable from version-controlled source.
- A pure-MATLAB integrator with identical maths
  (`scripts/etp_pure_matlab.m`) lets you cross-check the Simulink result
  and iterate ~20× faster.
